# Contributing to this project!

First off, thanks for taking the time to contribute! Whether you're a student, a researcher, or just interested in physics-informed machine learning, your input is welcome.

## How Can I Contribute?

### Reporting Bugs
If you find a bug in the PINN-former implementation or the metamaterial simulation scripts, please open an issue. Include:
* A brief summary of the problem.
* Steps to reproduce the behavior.
* What you expected to see vs. what actually happened.

### Suggesting Enhancements
I'm always looking for ways to make these models more efficient or accurate. If you have ideas for:
* New loss function terms for specific physics constraints.
* Optimization for hardware acceleration.
* Better data visualization for metamaterial lattice structures.
Please open an "Enhancement" issue to discuss it.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the documentation (README) is updated if your changes affect how the model runs.
4. Open a Pull Request with a clear description of your changes.

## Development Environment
To set up a local environment for testing:
1. `pip install -r requirements.txt`
2. `pip install pytest` (to run the test suite)

## Contact
If you have questions about the underlying physics or the research goals, feel free to reach out via GitHub issues or email.
